1、
`navigator.onLine`

2、
```
EventUtil.addHandler(window, "online", function(){
    alert("Online");
});
EventUtil.addHandler(window, "offline", function(){
    alert("Offline");
});
```

## 应用缓存

/offline.manifest中包含着描述文件，这个文件的MIME类型必须是text/cache-manifest
```
<html manifest="/offline.manifest">
```
