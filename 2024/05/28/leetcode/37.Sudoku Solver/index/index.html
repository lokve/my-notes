# 我的答案

记录每一行、每一列、每一个块中数字是否出现过，然后遍历空格，尝试填入 1-9，如果填入的数字合法，则递归调用 dfs，如果 dfs 返回 true，则说明填入的数字合法，继续填下一个空格，如果 dfs 返回 false，则说明填入的数字不合法，则尝试下一个数字，如果尝试完 9 个数字都失败，则说明整个数独不合法，返回 false。
