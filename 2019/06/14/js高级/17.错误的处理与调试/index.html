浏览器报告的错误

错误处理

try-catch

错误类型

  • Error
  • EvalError: eval 报错 eg: new eval();
  • RangeError: 数值超出范围 eg: var items1 = new Array(-20);
  • ReferenceError: 访问不存在的变量
  • SyntaxError: 在 eval 中使用错误语法 eg: eval(“a ++ b”)
  • TypeError: 类型错误 eg: var o = new 10;
  • URIError: encodeURI 传入格式不正确
