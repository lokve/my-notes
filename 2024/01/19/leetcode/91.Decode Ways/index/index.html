# 我的答案

尝试了这么多例子才出，已经搞懵了

这个题目本质是爬楼梯，但是要考虑 0 和 >26 的情况

  • 当 s[i] === 0 的时候，如果和前一个字母组成的数 >26 或者 =0 ，整个结果就是 0 了；不然他们只能是一个整体，所以 dp[i] = dp[i-2]
  • 当 s[i-1] === 0 时，他和前一个字母只能是一个整体，所以 dp[i] = dp[i-1]
  • 其他的时候就符合爬楼梯， dp[i] = dp[i-2] + dp[i-1]
  • 另外， i=1 的时候需要额外处理或者事先处理
