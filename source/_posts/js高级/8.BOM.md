## window

即使js访问浏览器的接口，也是ESCMScript的全局Global对象

### 全局作用域

```
var age = 29;
function sayAge(){
    alert(this.age);
}
alert(window.age); //29 
sayAge(); //29
 window.sayAge(); //29
```

使用var添加的全局属性的configurable默认为false, 所有不能`delete window.age`（无效）

### 弹出窗口

检测弹出窗是否被屏蔽

```
var blocked = false;
try {
        var wroxWin = window.open("http://www.wrox.com", "_blank");
        if (wroxWin == null){
            blocked = true;
}
} catch (ex){
    blocked = true;
}
if (blocked){
    alert("The popup was blocked!");
}
```
